# Installing Uffizzi

<Callout className="mt-6">
  **Note:** This guide describes how to install and configure the CLI for use with [Uffizzi Cloud](https://app.uffizzi.com) or a private managed version of [Uffizzi Enterprise](uffizzi.com/pricing). If you want to self-host open source Uffizzi, see the [Self-hosted Installation Guide](https://github.com/UffizziCloud/uffizzi/blob/develop/INSTALL.md) on GitHub.
</Callout>

<Steps>

### Download the Uffizzi CLI


<Tabs items={['Homebrew', 'Ruby Gem', 'curl']} defaultIndex="0">
    <Tabs.Tab>
        ```bash {10}
		brew tap uffizzicloud/tap && \
		brew install uffizzicloud/tap/uffizzi
		```
    </Tabs.Tab>
    <Tabs.Tab>
        ```bash {10}
		gem install uffizzi-cli
		```
    </Tabs.Tab>
    <Tabs.Tab>
        ```bash
        curl -L -o uffizzi "https://github.com/UffizziCloud/uffizzi_cli/releases/latest/download/uffizzi-darwin" && sudo install -c -m 0755 uffizzi /usr/local/bin && rm -f uffizzi
        ```
    </Tabs.Tab>
</Tabs>


### Configure the CLI with the default API server

Select the default server `app.uffizzi.com` (Uffizzi Cloud). Alternatively, if you're [self-hosting Uffizzi](https://github.com/UffizziCloud/uffizzi/blob/develop/INSTALL.md), use the Uffizzi API endpoint you configured.

```bash {10}
uffizzi config
# ...
# Server: (app.uffizzi.com)
```

### Login or create an account

Running this command will open a browser window where you can create an account or login to an existing account. If

```bash {10}
uffizzi login
# Login successful
# Select an account: (Press ↑/↓/←/→ arrow to move and Enter to select)
# ‣ jdoe
#   Acme Inc
```
</Steps>

That's it! You've successfully installed and configured the Uffizzi CLI.
