# `compose`

Manage [Docker Compose environments](/docs/core-concepts/ephemeral-environments/compose). This command was formerly called `preview`.

## `compose create`

Create a Docker Compose environment. See the [Uffizzi Compose Reference](compose-spec.md) for details.

``` bash {10}
uffizzi compose create [COMPOSE_FILE] [FLAGS]
```

| Flags                          | Description                                                                                |
|--------------------------------|--------------------------------------------------------------------------------------------|
| `--set-labels`                 | Set metadata for a deployment. Useful when filtering deployments                           |

To create a compose environment with single label, run:

``` bash {10}
uffizzi compose create docker-compose.uffizzi.yaml --set-labels github.repo=my_repo
```

## `compose delete`

Delete a Docker Compose environment given a valid compose ID. See the [Uffizzi Compose Reference](compose-spec.md) for details.

``` bash {10}
uffizzi compose delete [compose_ID]
```

## `compose describe`

Show metadata for a project given a valid preview ID. See the [Uffizzi Compose Reference](compose-spec.md) for details.

``` bash {10}
uffizzi compose describe [PREVIEW_ID]
```

## `compose list`

Lists all compose environments for a project, including active, building, deploying, failed, and sleeping.

``` bash {10}
uffizzi compose list [FLAGS]
```

| Flags                          | Description                                                                                |
|--------------------------------|--------------------------------------------------------------------------------------------|
| `--filter`                     | Metadata to filter list of deployments                                                     |
| `--output`                     | Format output as `json` or `pretty-json`                                                   |
