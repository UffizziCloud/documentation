# Using Uffizzi from the Dashboard 

The Uffizzi Cloud Dashboard provides a web-based user interface for managing your Uffizzi Cloud account. The Uffizzi Cloud Dashboard is available at [https://app.uffizzi.com](https://app.uffizzi.com).

From the dashboard, you can create and manage environments, teams, and project settings, as well as, managing your account settings and billing information.
