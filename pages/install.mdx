# Install Uffizzi

To begin, choose from the following Uffizzi options.

## Uffizzi Pro

With Uffizzi Pro you install the Uffizzi **data plane** your own Kubernetes cluster, while Uffizzi Cloud (`app.uffizzi.com`) acts as the control plane. Requests from the CLI are routed through Uffizzi Cloud, while workloads run on your cluster. [Learn more →](/install/pro#architecture)  

This option is recommended for most users since it provides the best balance of control and ease of use.

## Uffizzi Enterprise

Uffizzi Enterprise lets you install the Uffizzi **data plane** and **control plane** for a closed-loop configuration. This option is recommended for users who want maximum control or who require an air-gapped configuration. [Learn more →](/install/enterprise#architecture)
